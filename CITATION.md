# How to Cite

If this repository helps your project, report, or prototype, cite it as a
software benchmark repository and also cite the original tracking literature
for the specific algorithms you discuss.

## Suggested software citation

```text
Maneuvering Target Tracking Benchmark. Version 1.0.0.
GitHub repository, 2026.
```

## Suggested practice

- cite this repository for the public implementation and benchmark structure
- cite the classical CA, Singer, VD, IE, and IMM references for algorithm theory
- record the repository version or commit hash used in your experiments
