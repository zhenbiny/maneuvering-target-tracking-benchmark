# Presentation notes

This folder is intentionally kept lightweight in the public repository.

Private slide decks, defense-specific PDFs, and local presentation artifacts are
not tracked here. They should stay under `private_local/presentation/`.

For a public repository, the recommended approach is:

- keep only generic presentation notes or reusable figures here
- avoid committing files that contain personal names or local course metadata
- regenerate shareable slides separately if a public-facing deck is needed
