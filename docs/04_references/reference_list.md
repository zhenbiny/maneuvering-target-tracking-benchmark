# Reference List

This folder does not store downloaded paper PDFs in the public repository.
Instead, it keeps a lightweight index of the reference directions used in the
project.

## Algorithm families covered

1. Constant-acceleration Kalman filtering
2. Singer acceleration model tracking
3. Interacting multiple-model filtering
4. Variable-dimension maneuver tracking
5. Input-estimation-based maneuver tracking

## Recommended public practice

For a GitHub repository:

- keep citation information or a bibliography here
- do not commit copyrighted paper PDFs unless redistribution is clearly allowed
- document the algorithm origin in README files and source comments when needed

## Local-only reference storage

If you maintain downloaded papers locally, store them under:

- `private_local/reference_papers/`
