# Code of Conduct

This project aims to be welcoming, constructive, and technically focused.

## Expected behavior

- Be respectful and precise in technical discussions.
- Assume good intent and discuss ideas without personal attacks.
- Share criticism in a way that helps others improve the work.
- Keep discussions relevant to the repository and its benchmark scope.

## Unacceptable behavior

- Harassment, discrimination, or intimidation
- Personal attacks or insulting language
- Publishing private personal information
- Repeated off-topic disruption

## Maintainer guidelines

- Prioritize clarity, fairness, and a technically grounded discussion.
- Request changes on code, experiments, or documentation without targeting people.
- Remove or limit content that exposes private or sensitive information.

## Reporting

If a discussion becomes inappropriate, open a private channel with the
maintainers or, if needed, file a minimal issue describing the problem without
republishing harmful content.
